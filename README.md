# スーパーストア販売実績分析と改善施策の提案

## 1. プロジェクト概要
米国に展開するスーパーストア（小売店）の約1万件の販売データを用い、収益性改善のためのデータ分析を行いました。
Python（Pandas/Matplotlib）を使用して「売上と利益の相関」や「赤字要因」を可視化し、具体的なビジネス改善策を提案することを目的としています。

### 背景
- 大規模なデータセットから迅速に経営状況を把握する必要がある。
- 売上が高くても利益が伴っていないカテゴリを特定し、リソースの最適化を図る。

## 2. 分析結果（課題の特定）
データ分析を通じて、以下の3つの主要な課題を特定しました。

### ① カテゴリ別の収益格差
- **家電（Technology）**は売上・利益ともに高いが、**家具（Furniture）**は売上規模に対して利益が極端に低い。
- 家具カテゴリの中でも、特に「テーブル」が全体の利益を大きく押し下げている。

### ② 特定サブカテゴリの赤字問題
- **テーブル（Tables）**および**ブックケース（Bookcases）**において、販売すればするほど赤字が拡大する傾向が見られる。

### ③ 地域による利益構造の違い
- 同じ「テーブル」の販売でも、特定の地域（Central地域など）において赤字が深刻化しており、配送コストや地域限定の割引率に問題がある可能性が高い。

## 3. 改善提案
分析結果に基づき、以下の3点を提案します。

1. **価格戦略と割引率の見直し**:
   - 赤字商品（Tables等）に対する過度な割引を制限し、利益率を重視した価格設定へ変更する。
2. **高利益カテゴリへの集中投資**:
   - 利益率の高い「家電（Phones, Accessories）」および「事務用品（Paper）」のプロモーションを強化し、クロスセル（抱き合わせ販売）を促進する。
3. **物流・配送コストの最適化**:
   - 赤字が深刻な地域において、配送業者との契約見直し、または一定金額以下の注文に対する送料有料化を検討する。
